# 🎬 Flicked App – Flutter TMDB API Movie App  

🚀 **Flicked** is a Flutter-powered movie app that fetches real-time movie data from **The Movie Database (TMDB) API**. With a modern UI and interactive features, users can explore popular movies, search by category, and experience a **dynamic background** that changes based on the selected movie.  

## 🔥 **Key Features**  

✅ **Popular Movies Display** – Fetches real-time movie data from **TMDB API**  
✅ **Movie Search** – Easily find your favorite movies  
✅ **Movie Categories** – Browse by categories like **Popular, Now Playing, Upcoming, and Top Rated**  
✅ **Dynamic Background** – The **background changes** based on the selected movie's poster  
✅ **Modern & Responsive UI** – Sleek dark mode with a clean design  

## 🛠 **Technologies Used**  

- **Flutter** – UI framework for building cross-platform apps  
- **Dart** – Main programming language  
- **TMDB API** – API to fetch movie data  
- **Provider / Riverpod (Optional)** – State management for optimized performance  

## 🔗 **How to Run the App**  

1. Clone this repository  
   ```bash
   git clone https://github.com/AlbaniRajata/flicked_app.git
   cd flicked_app
   ```  
2. Install dependencies  
   ```bash
   flutter pub get
   ```  
3. Create an account on **TMDB** and get an API Key  
4. Store the API Key in an `.env` file or directly in the code as a **base URL**  
5. Run the application  
   ```bash
   flutter run
   ```  

## 💡 **Notes**  
- Ensure that **Flutter SDK** is installed and that you have an emulator or real device to run the app  
- The API Key must be configured before running the application
